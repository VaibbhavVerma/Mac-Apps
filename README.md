>> **Installation & Usage**

# <b>Awake:</b>
A simple macOS menu bar app that helps keep your Mac awake in one click.
1. Unzip the downloaded file.
2. Drag the app into the **Applications** folder.
3. Open the app. A **black eye** icon will appear in the menu bar.
4. Click the **black eye** to activate **Awake Mode**. The icon will change to a **colorful eye**, indicating that your Mac will stay awake.
5. Click the **colorful eye** again to disable Awake Mode and restore normal sleep behavior.

# <b>Noti Notch:</b>
Never miss a notification again. Get WhatsApp or any website notifications with a customizable colored border around your screen or MacBook notch.
1. Unzip the downloaded file.
2. Drag the **Noti Notch** app into the **Applications** folder.
3. Double-click the app to launch it.
4. Open the **Chrome Setup** tab.
5. Click **Download Extension** to save the Chrome extension to your Desktop.
6. Unzip the downloaded extension.

Install the Chrome Extension
1. Open **Google Chrome**.
2. Go to **Chrome > Settings > Extensions** (or visit `chrome://extensions`).
3. Turn **Developer Mode** **ON**.
4. Click **Load unpacked**.
5. Select the unzipped extension folder.

How It Works:
The Chrome extension creates a secure bridge between Chrome and the **Noti Notch** app.
Whenever Chrome receives a notification (such as from WhatsApp Web or any supported website), the extension instantly notifies the **Noti Notch** app. The app then displays a glowing border or notch effect on your screen, so you never miss a notification.

You can fully customize:
* Which edge of the screen glows (Top, Bottom, Left, or Right)
* Glow color
* Glow thickness
* Glow duration
